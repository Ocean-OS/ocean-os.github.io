JINST App Installer Builder v1.0

This is the official JINST file builder. This program lets you easily create a JINST file for your app. To start, just enter your app's information into the following inputs.



Enter your app's name. This will appear when the JINST file is first loaded.



Enter the name of the creators of the app (which is probably you or your company)



Enter the name of the folder your app will reside in. (eg MyApp)



Enter a URL that leads to your app's icon. This will appear when the JINST file is first loaded. Data URIs are allowed, but will increase the size of your JINST file.



Enter the year/date your app was created. This might appear when the JINST file is first loaded.





App Resources

The following section assumes you have uploaded your app's files to a website. If you have not done this, please do so before completing the form.

Windows (win32)

If your app is not compatible with Windows, select the checkmark.

Enter the URL that leads to a zip or exe file for your app, if it were running on an arm64 processor on Windows.



Enter the URL that leads to a zip or exe file for your app, if it were running on an ia32 processor on Windows.



Enter the URL that leads to a zip or exe file for your app, if it were running on an x64 processor on Windows.



MacOS (darwin)

If your app is not compatible with MacOS, select the checkmark.

Enter the URL that leads to a zip or dmg file for your app, if it were running on an arm64 processor on MacOS.



Enter the URL that leads to a zip or dmg file for your app, if it were running on an x64 processor on MacOS.



Linux (linux)

If your app is not compatible with Linux, select the checkmark.

Enter the URL that leads to a zip or jsar file for your app, if it were running on an arm64 processor on Linux.



Enter the URL that leads to a zip or jsar file for your app, if it were running on an x64 processor on Linux.
